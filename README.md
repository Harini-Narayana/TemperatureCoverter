# Temperature Converter

A simple and interactive temperature converter website that allows users to convert temperatures from Celsius to Fahrenheit and Fahrenheit to Celsius with ease. The project focuses on clean design, smooth functionality, and a user-friendly experience.

Built using HTML, CSS, and JavaScript, the website provides accurate conversions through a responsive and visually appealing interface.

## Features
- Celsius to Fahrenheit conversion
- Fahrenheit to Celsius conversion
- Simple and responsive interface
- Fast and accurate calculations
- Clean and modern design

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Purpose
This project was created to strengthen frontend development skills by combining JavaScript logic with responsive web design.
